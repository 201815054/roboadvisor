주가 분석 사용 모델



[시계열 모델 (Time Series Models)]

<장점>
시계열 데이터의 패턴과 추세를 고려하여 예측하기 때문에 시간적 의존성을 잘 반영할 수 있음
과거 데이터만을 사용하기 때문에 예측 결과가 해석 가능하고 설명력이 높을 수 있음
<단점>
시계열 모델은 일반적으로 단변량 데이터에 적용되기 때문에 다양한 요인을 고려하기 어려움
정확한 예측을 위해서는 데이터의 정상성(stationarity)을 확인하고 전처리해야 할 수도 있음

1.ARIMA (Autoregressive Integrated Moving Average)
- 자기회귀(AR), 차분(Integrated), 이동평균(Moving Average)의 세 가지 요소로 구성. 
- ARIMA는 과거 관측치의 선형 조합을 사용하여 예측을 수행합
2.SARIMA (Seasonal ARIMA)
-  SARIMA는 ARIMA 모델의 계절성 요소를 고려한 모델
-  시계열 데이터에서 계절성 패턴을 반영하여 예측
3.Prophet
- Prophet은 Facebook에서 개발된 시계열 예측 라이브러리로, 일일 데이터와 계절성 요소를 고려하여 예측
- 휴일 등의 특별한 이벤트도 고려할 수 있음



[회귀 모델 (Regression Models)]

<장점>
주가 예측에 사용할 수 있는 다양한 변수를 고려하여 모델링할 수 있음
과거 데이터와 다른 요인들 간의 관계를 통해 예측을 수행하기 때문에 설명력이 높을 수 있음
<단점>
주가와 관련된 다양한 변수들 간의 비선형적인 관계를 반영하기 어려울 수 있음
모델의 복잡성이 증가하면서 과적합(overfitting) 문제가 발생할 수 있음

1. 선형회귀
2. 다항회귀


[신경망 모델 (Neural Network Models)]

<장점>
LSTM과 같은 신경망 모델은 장기적인 시계열 의존성을 학습하기에 용이
비선형적인 관계를 모델링할 수 있고, 다양한 변수를 함께 고려할 수 있음
<단점>
대량의 데이터와 복잡한 모델 구조를 필요로 하며, 학습 및 예측에 시간과 계산 자원이 많이 소요될 수 있음
신경망 모델의 해석이 어렵고, 과적합 문제에 취약할 수 있음

1.LSTM (Long Short-Term Memory)
- 순환 신경망(RNN)의 한 종류로, 시계열 데이터의 장기적인 의존성을 학습하는 데 효과적
- 기억 셀과 게이트 메커니즘을 사용하여 시간적인 패턴을 기억하고 예측합니다.
2.변이형 오토인코더 (Variational Autoencoder)
- 비지도 학습 모델로, 데이터를 압축하고 잠재 공간으로 표현하여 주가의 잠재적인 구조를 학습하고 이후 잠재 공간에서 새로운 샘플을 생성하거나 예측에 사용할 수 있음
3.생성적 적대 신경망 (Generative Adversarial Networks)
- 생성적 적대 신경망은 생성자와 판별자라는 두 개의 신경망을 경쟁시켜 원하는 출력을 생성하는 모델
- 주가 예측에는 주식 데이터의 분포를 학습하고 이를 기반으로 새로운 샘플을 생성하는 데 사용될 수 있음


[앙상블 모델 (Ensemble Models)]

<장점>
다양한 예측자들을 조합하여 예측하기 때문에 예측 성능을 향상시킬 수 있음
다양한 알고리즘을 사용할 수 있기 때문에 각 모델의 강점을 살릴 수 있음
<단점>
모델들을 학습하고 예측하기 위한 계산 비용이 높을 수 있음
모델들 간의 상호작용을 고려해야 하며, 앙상블의 구성이 복잡해질 수 있음

1.랜덤 포레스트 (Random Forest)
- 결정 트리(Decision Tree)를 기반으로 하는 앙상블 모델
- 다수의 결정 트리를 조합하여 예측을 수행하며, 주가 예측에는 다양한 변수들의 중요도를 파악할 수 있음
2.그래디언트 부스팅 (Gradient Boosting)
- 그래디언트 부스팅은 여러 개의 결정 트리를 순차적으로 학습하여 예측 성능을 향상시키는 앙상블 모델 
- 이전 트리의 오차를 보정하는 방식으로 학습하며, 주가 예측에 강력한 예측 성능을 제공할 수 있음
